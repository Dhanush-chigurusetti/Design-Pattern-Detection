# ML-Based Software Design Pattern Detection

## 📌 Overview
This project presents an automated approach for detecting software design patterns in source code using machine learning techniques. It addresses the challenges of identifying patterns due to variations in coding styles and programming languages.

## 🚀 Features
- Automated detection of software design patterns
- Hybrid deep learning model (CNN + RNN)
- Structural and behavioral feature extraction
- High accuracy and cross-language generalization

## 🛠️ Tech Stack
- Python
- TensorFlow / Keras
- Machine Learning & Deep Learning

## 📊 Methodology
- Preprocessed and annotated source code dataset
- Extracted meaningful features from code
- Trained CNN + RNN model for classification
- Evaluated using accuracy, precision, recall, and F1-score

## 🎯 Applications
- Code comprehension
- Software maintenance
- Refactoring support
- Developer productivity improvement

## ▶️ How to Run
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the main file: `python main.py`

## 📈 Results
The model achieves high accuracy and generalizes well across multiple programming languages.
